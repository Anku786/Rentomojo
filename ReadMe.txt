How to run the Phone Directory  Project
1.Download the zip file
2. Extract the file and copy the folder
3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name renotmojo
6. Import renotmojo.sql file(given inside the zip package)
7.Run the script http://localhost/Renotmojo/homePage.php for the Phone directory
8.Run the script http://localhost/Renotmojo/addPhoneNumber.php to Add Phonenumber
9.Run the script http://localhost/Renotmojo/EditPhoneNumber.php to Edit Phonenumber

10.Run the script http://localhost/Renotmojo/homePage.php for the Updated Phone directory
